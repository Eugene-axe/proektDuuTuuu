Класс User : 
    - хранит в себе имя, логин, пароль, массив задач
    - умеет: - создавать новую задачу
             - сохранять новую задачу себе в архив
             - отправлять свой архив задач в LocalStorage под ключем Login
             - вынимать из LocalStorage свои задачи, по ключу своего Logina

Класс Task: 
    - конструктор задач
    - умеет: - менять статус задачи на "завершенный" и определять дату завершения

Main: основной исполнительный документ
    свойство: handler - сдержит обработчики временных(удаляемых) слушателей 
              modal - хранит возникающие модальные окна, чтобы удалять можно было

    currentUser() - определяет текущего пользователя
    headerGreeting() - меняет хэдер относительно текущего пользователя
    asideHandler() - события сайдбара: 
        выдвигается сайдбар по нажатию стрелки
        Кнопка добавить новую задачу вызывает модалку getCreateModalAddNewTask()
    getCreateModalAddNewTask() - выводит на экран модалку добавления новой задачи
    createTaskGrid(user) - создаст сетку из задач юзера
    createTaskStick() - создает блок стикер задачи
    run() - запуск приложения
